Project 1: Automated Test Framework for Accuload Systemp

This project involves building a fully automated test framework designed to run a series of transactions on the Accuload system. It uses a central ScriptEngine.py file that processes test cases via YAML configuration files. The framework handles complex scenarios involving multiple test scripts, providing sequential control over transactions for different arms in the Accuload system.

Technologies used:Python, API, YAML for configuration management

Challenges Faced

Complexity in Transaction Management: Managing multiple transactions concurrently on different arms of the Accuload system was one of the primary challenges. Each arm needed to run a batch process sequentially, but the framework had to ensure no overlap or premature triggering of new transactions before the previous one had completed.

Learnings | Problem Solving Skills

Through this project, I developed strong problem-solving skills, particularly when working with complex system integrations. I learned how to break down a large problem into smaller, manageable components (e.g., handling each arm's transaction individually, managing batch completion statuses).

Automated Testing Framework for Batch Process Management

Developed an automated testing framework to manage and monitor batch processes across multiple arms in interconnected bays, using the Accuload API. The framework automates batch allocation, status monitoring, and retries, ensuring synchronization and improving testing efficiency in complex systems.

View Demo

Technologies used: Python, Accuload API, Logging, Automation, Testing

Challenges Faced

Ensuring synchronization between multiple interconnected arms and managing batch retries in real-time. Handling complex interactions within the testing setup, where ending a batch in one arm affected the others, required custom logic to monitor and control the flow effectively.

Learnings

Improved understanding of API integration, batch process management, and automated testing techniques. Gained valuable experience in debugging complex systems and implementing retry mechanisms for robustness in critical processes.
